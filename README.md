# front_game
## Four in Line - «4 в ряд»

### Описание

Это веб-версия классической настольной игры "4 в ряд".

Играют два человека - красные (игрок 1) и жёлтые (игрок 2).

Цель игры - собрать четыре фишки подряд по горизонтали, вертикали или диагонали.


### Как играть

1. Каждый игрок по очереди нажимает на кнопку над колонкой, чтобы бросить туда фишку.
2. Фишка падает вниз в самую нижнюю свободную ячейку.
3. После каждого хода ход переходит к другому игроку.
4. Побеждает тот, кто соберёт 4 фишки подряд.
5. Если всё поле заполнено и никто не выиграл - объявляется ничья.

### Управление

* Новая партия - начать игру заново.
* Сбросить таблицу - очистить результаты и начать с нуля.

### Запуск проекта

1. Установите зависимости:

   ```bash
   npm install
   ```
2. Запустите игру:

   ```bash
   npm run dev
   ```
3. Откройте браузер и перейдите по ссылке

После этого можно играть прямо в браузере.
